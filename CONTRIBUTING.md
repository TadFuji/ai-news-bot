# Contributing to AI News Bot

ご興味を持っていただきありがとうございます！
バグ報告や機能追加の提案、プルリクエストは大歓迎です。

## 開発フロー

1. このリポジトリをForkしてください
2. 新しいブランチを作成してください (`git checkout -b feature/amazing-feature`)
3. 変更をコミットしてください (`git commit -m 'Add some amazing feature'`)
4. ブランチにPushしてください (`git push origin feature/amazing-feature`)
5. Pull Requestを作成してください

## 環境構築

```bash
# クローン
git clone https://github.com/YourUsername/ai-news-bot.git
cd ai-news-bot

# 依存関係のインストール
pip install -r requirements.txt

# 環境変数の設定 (.env)
# GOOGLE_API_KEY などが必要です
```

## コードスタイル
- Pythonの標準的なコーディング規約（PEP 8）に準拠してください。
- コメントやコミットメッセージは日本語で記述することを推奨します。
